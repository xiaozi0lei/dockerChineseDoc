# dockerChineseDoc
**Docker Chinese Translation Docs**

***

##Goal

此项目希望能帮助到喜欢docker技术的人们，同时希望更多的人们了解docker的技术。
希望我们能为国内的IT技术尽自己的微博之力。

下面按照Docker官方网站的布局，如果国内github访问太慢，可以选择后面的博客同步更新。
